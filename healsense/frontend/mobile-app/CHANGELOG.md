# Changelog

All notable changes to the HealSense Mobile App are documented here.

## [0.0.1] - 2026-01-21

### Added
- Initial React Native app with Expo
- User authentication system (login/signup)
- Real-time vital signs monitoring dashboard
- Alert system for abnormal vital readings
- Profile management with edit capabilities
- Medical conditions and medications tracking
- Emergency contact management (Doctor & Family)
- Emergency dialog with call and WhatsApp integration
- Dark mode support across all screens
- Session persistence with AsyncStorage
- Clear all alerts functionality
- Bottom tab navigation (Dashboard, Alerts, Profile, Settings)

### Features
- **Dashboard**: Real-time display of 6 vital signs with color-coded status
- **Alerts**: List of all health alerts with acknowledge/dismiss options
- **Profile**: View and edit personal information, medical history, and emergency contacts
- **Settings**: App preferences including dark mode toggle and logout
- **Authentication**: Secure login/signup with form validation
- **Emergency System**: Quick access to emergency contacts with formatted WhatsApp messages

### Technical
- TypeScript for type safety
- React Navigation for routing
- React Query for data fetching
- Context API for global state
- AsyncStorage for local persistence
- Expo Vector Icons for UI
